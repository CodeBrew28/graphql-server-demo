
# GraphQL Server Examples

## Setup

Installing packages 
```
npm install
```

Creating the database and reseting if there are any changes 
```
./bin/reset
```

Starting the server
```
npm run start 
``` 

Head to https://studio.apollographql.com/dev 

Create the project and set the endpoint to http://localhost:4000/graphql
